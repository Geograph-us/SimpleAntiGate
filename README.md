SimpleAntiGate
==============

Простой класс для работы с сервисом антикапчи AntiGate.com на C#
Для использования класса достаточно бросить его файл SimpleAntiGate.cs (13 Кб) рядом со своими исходниками и подключить его через using.

Модуль может распознавать капчу из файла, по ссылке, из массива байт (byte[]), из потока (Stream) и из объекта типа Image.

Описание функций:
Recognize – умеет распознавать картинку используя сервис AntiGate.com из потока, массива байт и Image, по URL-ссылке и из картинки на компьютере.
GetBalance – выводит текущий баланс на сервисе AntiGate.com
ReportBad – отправляет жалобу о неверно распознанной капче на сервис AntiGate.com

В архиве с классом прилагаются примеры его использования.
